Finding Algoraves

livecoding music performance algorave

this is me ... playing my set during an algorave for the first time..

but let's go back a bit, because a few months ago i didn't even know algoraves existed..

last fall i visited my first one in Amsterdam. an algorave is a party where music and visuals are performed with code. and for someone like me, who for some strange reason loves code, there was something immediately special about it.

the code is displayed on screens during the performance.. and if a program crashes... everyone starts clapping. at first i didn't get it.. like... why? what's the point?

but then i realized that these details create a shared language between everyone in the room, like a shared understanding of how code works, and how code itself creates the atmosphere of the party.

and that's exactly what brought together 2 sides of myself i'd rather have kept separate: being a developer and being a musician..
