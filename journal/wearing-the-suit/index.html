Wearing the Suit

working with AI tools

after using cursor and switching between codex and claude since 2024, i can clearly see there's a difference between building with blanche and trying to get blanche to build for you... the 1st feels like wearing the "iron man" suit, the 2nd feels like we're already in part 3, building the "iron legion"... but are we really there?

i was convinced i'd ship everything i'd ever wanted to build, all these new features, side projects, the ideas that had been waiting. what actually happened was slower and stranger..

i got good at filling in the .md blanks... got fast at generating code that almost worked. and i spent a lot of time in a weird space where things were "nearly ready" but not ready for real production use... the promise is: fill the docs carefully enough and blanche generates 99% of the code, you add your 1% cherry on top and ship. what nobody says: instead of building, you're briefing. rewriting the same context, wondering if it's enough. and every prompt, you never know if it works until it doesn't.

i don't think blanche tools are hype... i think handing over authorship and hoping for great code to arrive is naive. the tool that generates code still needs someone who knows what the code is for, who can tell good from almost-good, and who will stand behind it when something breaks...

so this year i'm trying something different... wear the suit..

not the iron man version with autonomous robots managing the fight while he watches.. that's part 3 stuff... it comes after you already understand how the suit works...

i mean the mechanic's version... heavy, full of tools, start from my own understanding, use blanche to move faster and think harder, but keep the judgment mine...
