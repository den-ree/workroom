# Den Ree Workroom

Portfolio site for **Den Ree** — tech founder and musician. The site presents my apps, music work and my journal. Built as a static site with HTML, CSS, and JavaScript; contact form handled by Formspree. Deploys to GitHub Pages on push to `main`.

## Run locally

Open the project folder and serve it (e.g. with a local server). Example with Python:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Structure

- `index.html` — Home (hero, apps, music, contact)
- `contact/index.html` — Contact
- `music/index.html` — Music portfolio
- `journal/index.html` — Journal
- `css/`, `js/`, `images/` — Assets
